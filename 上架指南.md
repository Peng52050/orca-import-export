## 待完成：提交 PR 到 awesome-orcanote

### 步骤 1：Fork 仓库

访问 https://github.com/sethyuan/awesome-orcanote ，点击右上角 **Fork** 按钮。

### 步骤 2：编辑 plugins.json

1. 进入你 fork 的仓库：`https://github.com/Peng52050/awesome-orcanote`
2. 点击 `plugins.json` 文件
3. 点击右上角铅笔图标 ✏️ 编辑
4. 找到 `SaXz2` 的 `bullet-theme` 条目之前（按 author 字母顺序，`Peng52050` 应插入在 `P` 区段，在 `litcu` 之后）
5. 在 `litcu` 的 `task-planner` 条目后面，插入以下 JSON（注意逗号）：


### 步骤 3：创建 Pull Request

1. 回到你 fork 的仓库主页
2. 点击 **Contribute** → **Open pull request**
3. 填写 PR 信息：

**Title**: `Add orca-import-export plugin`

**Body**:
```
## 插件信息

- **名称**: Orca 导入导出
- **功能**: 多格式导入导出插件，支持 Logseq/Obsidian/Orca Note 高亮语法转换

## 功能特性

### 导入功能
- 支持 .md / .txt / .html / .opml / .json 格式
- 自动检测文件格式
- 高亮语法自动转换为 Orca 原生格式
- 挖空语法支持

### 导出功能
- 支持 Markdown / HTML / Word / OPML / JSON 格式
- 多种导出样式：Logseq / Obsidian / Orca Note / SiYuan

### 挖空功能
- 导入时支持多种挖空语法识别
- 导出时可转换为指定挖空格式

## 质量标准

- ✅ 功能完整，已测试
- ✅ 包含 README 文档
- ✅ MIT 开源许可证
- ✅ 活跃维护

## 仓库链接

- 项目主页: https://github.com/Peng52050/orca-import-export
- 下载链接: https://github.com/Peng52050/orca-import-export/releases/download/v2.4.7/orca-import-export_v2.4.7.zip
```

4. 点击 **Create pull request**
