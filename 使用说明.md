## 插件概述
**orca-import-export** 是一款为 Orca Note 设计的多格式导入导出插件，支持在 Orca Note 与其他笔记应用（Logseq、Obsidian、思源笔记等）之间进行数据迁移，同时支持多种文档格式（Markdown、HTML、Word、JSON、OPML）的导入导出。
### 核心特性
- ✅ **多格式互转**：支持 Logseq / Obsidian / Orca Note / SiYuan 四种笔记应用格式，以及 Markdown、HTML、Word、JSON、OPML、TXT 六种文件格式
- ✅ **富文本双向转换**：支持高亮、加粗、斜体、删除线等富文本样式的导入和导出
- ✅ **挖空模式**：支持多种挖空语法与 Orca 挖空效果的双向转换
- ✅ **表格支持**：保持表格结构在导入导出过程中的完整性
- ✅ **图片支持**：支持 Markdown 图片语法 `![](url)` 的导入和导出
- ✅ **进度反馈**：导入导出过程中显示实时进度，支持取消操作
- ✅ **实时预览**：导入前可预览转换结果，支持编辑调整

## 安装与卸载
### 安装
1. 下载最新版本的插件包（`.zip` 文件）
2. 在 Orca Note 中打开 **设置 > 插件**
3. 点击 **安装插件**，选择下载的 `.zip` 文件
4. 插件自动加载，工具栏会出现导入/导出按钮
### 卸载
1. 在 Orca Note 中打开 **设置 > 插件**
2. 找到 **orca-import-export**，点击右侧的 **卸载**
3. 插件自动清理并卸载

## 快速开始
### 导入文件
1. 在工具栏点击 **导入** 按钮，或使用快捷键 `Ctrl+Shift+I`
2. 选择要导入的文件（支持 `.md`、`.txt`、`.opml`、`.xml`、`.json`）
3. 在预览面板中调整导入设置（格式、挖空模式、富文本转换等）
4. 点击 **导入**，选择插入位置（子块或同级）
5. 等待进度完成，内容自动插入到当前文档
### 导出文件
1. 在工具栏点击 **导出** 按钮，或使用快捷键 `Ctrl+Shift+E`
2. 选择导出样式（Logseq / Obsidian / Orca Note / SiYuan）
3. 选择导出格式（大纲类 / Markdown 类 / 文档类）
4. 调整导出设置（最大深度、富文本转换、挖空模式等）
5. 点击 **导出**，文件自动下载

## 导入功能
### 导入方式
插件支持三种导入方式：
| 方式 | 说明 | 操作 |
|------|------|------|
| **文件选择** | 点击"选择文件"按钮 | 点击导入对话框中的文件选择按钮 |
| **拖放导入** | 将文件拖放到对话框的拖放区域 | 从文件管理器拖放文件到对话框 |
| **剪贴板粘贴** | 粘贴剪贴板中的文本内容 | 使用 `Ctrl+V` 粘贴到对话框的源码编辑器 |
### 导入设置
导入对话框提供以下设置选项：
#### 基础设置
- **导入格式**：自动检测或手动选择源格式（Logseq、Obsidian、Orca、SiYuan、纯文本）
- **插入位置**：作为子块插入（推荐）或作为同级兄弟插入
#### 样式设置
- **转换富文本语法**：启用后将源格式的高亮语法转换为 Orca 富文本格式
- **挖空模式**：启用后将指定语法转换为 Orca 挖空（遮挡）效果
### 挖空模式
挖空模式允许将特定语法标记的内容转换为 Orca 的挖空（遮挡）效果。
#### 支持的挖空语法
| 语法 | 值 | 说明 |
|------|-----|------|
| `〖内容〗` | `tortoise` | 龟兔赛跑风格挖空 |
| `[[c1::xx]]` | `cloze-idx-bracket` | 带索引的方括号挖空（Anki 格式） |
| `[[xx]]` | `bracket` | 方括号挖空 |
| `{{xx}}` | `brace` | 花括号挖空 |
| `**xx**` | `bold` | 粗体挖空 |
| `***xx***` | `bold-italic` | 粗体+斜体挖空 |
| `*xx*` | `italic` | 斜体挖空 |
| `"xx"` | `quote` | 引号挖空 |
### 富文本转换
启用"转换富文本语法"后，插件会自动识别并转换以下富文本样式：
| 源格式 | 高亮语法 | 转换后效果 |
|--------|----------|------------|
| Logseq | `[[#red]]==text==` | Orca 背景高亮（红色） |
| Obsidian | `<span style="background:#ff4d4f">text</span>` | Orca 背景高亮（红色） |
| SiYuan | `<span data-type="backgroundColor">text</span>` | Orca 背景高亮（对应颜色） |
| Orca | 原生高亮 | 保持原样 |
支持的高亮颜色：红色、蓝色、绿色、黄色。

## 导出功能
### 导出样式
| 样式 | 说明 | 高亮语法 |
|------|------|----------|
| **Logseq** | Logseq 风格格式 | `[[#red]]==text==` / `[[$red]]==text==` |
| **Obsidian** | Obsidian 风格格式 | `<span style="background:...">text</span>` / `<span style="color:...">text</span>` / `==text==` |
| **Orca Note** | Orca 原生风格 | 原生高亮格式 |
| **SiYuan** | 思源笔记风格格式 | `<span data-type="backgroundColor">text</span>` |
### 导出格式分组
#### 大纲类
- 大纲（无序列表）
- 大纲（有序列表）
- 任务列表
- 树形大纲
#### Markdown 类
- {样式名} Markdown（Obsidian/Logseq/Orca/SiYuan）
#### 文档类
- HTML（大纲）
- Word（.doc）
- JSON
- OPML
- 纯文本
### 导出设置
#### 基础设置
- **最大深度**：导出时的层级深度（H1~H10，默认为 H1~H6）
#### 样式设置
- **转换富文本语法**：启用后将 Orca 富文本格式转换为目标格式的语法
- **挖空模式**：启用后将 Orca 挖空效果转换为指定语法
#### Word 专用设置
- **Word 标签模式**（仅 Orca + Word）：
  - **传统标签**：使用 `<mark>` 和 `<span style>` 标签
  - **Orca 原生**：使用 `<span class="orca-inline">` 标签

## 设置面板
插件提供全局设置，在 Orca Note 的 **设置 > 插件 > orca-import-export** 中访问：
### 版本徽章
设置面板顶部显示版本徽章，包含：
- 插件名称和版本号（不可编辑）
- 插件描述
### 设置项
| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| **调试开关** | 布尔 | `false` | 开启后在控制台输出详细日志 |
| **默认导出样式** | 单选 | `orca` | 打开导出对话框时的默认样式 |
| **默认转换富文本语法** | 布尔 | `true` | 打开导出对话框时"转换富文本语法"的默认状态 |
| **默认最大深度** | 数字 | `6` | 导出时的默认层级深度 |
| **导入插入位置** | 单选 | `child` | 导入内容的默认插入位置 |
| **挖空模式** | 布尔 | `true` | 导入时是否默认启用挖空模式 |
| **挖空语法** | 多选 | `['bold']` | 导入时默认识别的挖空语法 |
| **导出挖空模式** | 布尔 | `false` | 导出时是否默认启用挖空模式 |
| **导出挖空语法** | 多选 | `['bold']` | 导出时默认转换的挖空语法 |

## 常见问题
### Q1：导入后图片无法显示？
A：请检查图片链接是否有效，在线图片需要网络连接。
### Q2：导入后表格格式错乱？
A：Orca 的表格结构与 Markdown 不同，复杂表格可能需要手动调整。
### Q3：挖空效果不生效？
A：请检查是否启用了"挖空模式"并选择了正确的挖空语法。
### Q4：导出的 Word 文件样式不正确？
A：请尝试切换"Word 标签模式"。
### Q5：如何取消正在进行的导入/导出？
A：在进度浮层中点击"取消"按钮。
